Sitio personal de Paula Pinzón Maldonado — especialista en IA Generativa, conferencista y consultora en gobernanza de IA.

Contenido principal:

- Página principal: `index.html`

Instalación y desarrollo local

```bash
cd paulaandreapinzon-main
npm install
npm run dev
```

Publicar en GitHub

1. Crea un repositorio en GitHub (por ejemplo `pauandreapinzon/Pagina-web-Paula-Pinzon`).
2. Inicializa git en la carpeta del proyecto o usa GitHub Desktop.
3. Añade, commitea y push:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/pauandreapinzon/Pagina-web-Paula-Pinzon.git
git push -u origin main
```
